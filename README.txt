-------------------------------------------
Hypixel Nick Sniper
------------------------------------------
This project gets rid of the annoying repetitiveness of manually looking for nicks.


-------------------------------------------
How to use
------------------------------------------
SETUP
- Download the newest version of the mod
- Download Forge for 1.8.9 
- Drag mod into %appdata%\.minecraft\mods
- Launch Forge and connect to mc.Hypixel.net

Useage
- For optimal use go to bedwars lobby
- Go to the swimming pool diving board to not get afk'ed
- https://i.imgur.com/fLYGOhj.png
- run /nick help setrandom
- to stop just press escape (maybe more than once)

If you find a nick you like either click accpet Or run /nick actuallyset (Name)


-------------------------------------------
"Good nick" Criteria
------------------------------------------
- No underscores
- No numbers
- 9 or less Characters
- No names

to change to edit this you can add |bob to the end after tobi
